# Gene Keys: Complete 64-Gate Reference
# Shadow → Gift → Siddhi for all 64 gates
# Based on Richard Rudd's Gene Keys system, derived from the 64 I Ching hexagrams.
#
# Usage in synthesis: Focus on the Conscious Sun gate (= Life's Work),
# Conscious Earth gate (= Life's Anchor), and any gates forming defined channels.
# Shadow = contracted/fear-based expression | Gift = integrated talent | Siddhi = highest potential

## How to Map Gates to a Chart
# In Human Design, each planet activates two gates: Personality (conscious) and Design (unconscious).
# The HD API returns the full gate list. The most important for synthesis:
# - Personality Sun gate = Gene Key 1 (primary life theme, most conscious)
# - Personality Earth gate = Gene Key 2 (grounding, support for the Sun)
# - Design Sun gate = Gene Key 3 (unconscious gifts, often seen more by others)
# - Any gate forming part of a defined channel = active Gene Key

## Using This Reference in Synthesis

1. **Identify the top 4-6 gates** from the HD chart (Conscious Sun, Conscious Earth, Design Sun, Design Earth, plus any gates in defined channels)
2. **Name the shadow** as the specific behavioral pattern, not an abstract quality
3. **Name the gift** as a practical capability visible in real life
4. **Connect to chart placements** — which house does this gate activate? What does the house domain add?
5. **Cross-reference** — does this gate's theme appear in BaZi element analysis, Vedic nakshatra, or numerology Life Path?
6. **Ask the transformation question** — "What would it look like if this gift were fully active right now?"
